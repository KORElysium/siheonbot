** 무조건 config.json 안에 토큰과 접두사를 적어주셔야 합니다 **

---- 폴더 안내 ----

KS BOT
기본 봇 폴더입니다
건드리시면 봇이 정상적으로 작동하지 않습니다

Config
봇의 설정이 담긴 폴더입니다
봇 토큰이나 프리픽스, 유저 아이디 같은 정보를 넣을 수 있습니다

Commands
커맨드가 담긴 폴더입니다
파일을 추가하거나 수정하실 수 있으며
자유롭게 사용 가능하나 폴더 이름은 바꾸시면 안 됩니다

---- 파일 안내 ----

index.js
봇의 CPU가 되는 폴더입니다
이 파일은 건드리지 않으셔도 됩니다

config.json
봇의 프리픽스(접두사)와 토큰을 넣는 파일입니다
따옴표 안에 토큰과 원하시는 프리픽스를 넣으시면 됩니다

LICENSE.txt
봇의 저작권(라이센스) 파일입니다

package-example.json
설치할 모듈이 있는 폴더입니다
봇을 실행하실때는 npm init -y 로 새로 생성 해주셔야 합니다

start.bat
이 파일을 더블클릭 하시면 봇을 실행 시키실 수 있습니다
